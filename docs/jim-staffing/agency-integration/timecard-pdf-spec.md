## Weekly Timecard PDF (Employee Portal) — Spec

Prepared by: **Joel S. Premier — Senior Software Engineer**

This spec describes the employee-authenticated weekly timecard PDF generated by:
- `GET /api/v1/staffing/my-times/export.pdf?week=this|last`

### Layout rules
- One page only
- 8 rows: 7 days (Sun–Sat) + 1 totals row
- Letter size (8.5×11)

### Row fields (per day)
- Day + Date
- Shift (DAY/NIGHT)
- Time In / Time Out
- Lunch defaults to 30 minutes when work exists
- Hours total minus lunch
- Verified via (Wi-Fi / Location / transitions)
- Signature image thumbnail when signed

### Data rules
- Pair CLOCK_IN → CLOCK_OUT across the full Sun–Sat window (handles cross-midnight)
- Roll up hours into the day of CLOCK_IN
- Shift classification stored on clock-out when possible
